Day 1 概念介绍

简介：

  • 什么是ML？ data（资料）——>ML——>skill（某种表现的增进）。授电脑以渔。
  • 为什么用ML？an alternative route to build complicated systems。（自己学习自己分析然后得到结果）
  • ML应用？无法人工编程的；无法简单定义解的；需要快速决策的（几秒内，人类无法做到）；大规模的user-oriented
  • 什么时候用/关键要素？
    exists some underlying pattern to be learned
    no programmable(easy) definition
    data
    (有规则、不知道如何使用、有资料）

符号化表示：

  • unknow pattern to be learned——>target function | f
  • data——>training examples：D
  • hypothesis set ： H （很多可能的模式集合）
  • learning alg A（演算法，从H中找一个最接近f 的g）
  • learning model = A and H

机器学习：use data to compute hypothesis g that approximates target f （从资料出发，算出一个假说g，让其与 f 最接近）

机器学习和数据挖掘、人工智能、统计学：
Data Mining：use (huge) data to find property that is interesting.（很难区分，但不完全相同）
Artificial Intelligence：compute something that shows intelligent behavior. (ML是实现AI的方式之一）
Statistics： use data to make inference about an unknown process.（ML的工具）

Day 3

  • binary classification：二元分类。常用于垃圾邮件判断、发不发信用卡、是否生病、广告是否盈利等。
  • Multiclass Classification：硬币识别问题；图片识别；电子邮件类别。especially for recognition。
  • Regression：（bounded regression）：预测股票价格；预测天气等。（很多统计上的工具可以在这部分使用）。输出空间无限大的实数。
  • Structured learning：结构化学习。训练集有多种结构。
  • 等等
  • supervised：监督式学习（以硬币识别为例，提前知道有哪几种重量的硬币表示多少钱）
  • Unsupervised：无监督学习（自己聚类，不知道几种。）应用：文章分类、客户分类等。
  • Semi-supervised：半监督。有部分标记点。（问题特点：找到标记点很贵）
  • Reinforcement Learning：强化学习。（如广告推荐系统，用另外一个输出动作的好坏来学习，好奖赏，不好惩罚）
